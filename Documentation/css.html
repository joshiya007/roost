Page Headers

Page header is a wrapper for page titles and optional page actions.

Basic Page Header

  • Front-end
  • Dashboard
<section class="section">
    <div class="container">

        <!-- page header starts -->
        <header class="section__title">
            <h1>8,243 Properties for Sale, Rent & Real Estate</h1>
            <small>Villas, Apartments, Office Spaces and Commercial Builsings</small>
        </header>
        <!-- page header ends -->

        <!-- page content -->

    </div>
</section>
<section id="main__content">
    <div class="main__container">

        <!-- page header starts -->

Containers

Roost requires a containing element to wrap site contents in order to make it center aligned. There are 2 different variant of containers available in Roost for both front-end and dashboard templates. Note that, due to padding and more, neither container is nestable.

Front-end

Use .container for a responsive fixed width container.

<div class="container">
    <!-- contents -->
</div>

Dashboard

Use either.main__container or .main__container .main__container-sm for a responsive fixed width container. .main__container-sm is the smaller modifier of the main one and it will limit the width to 900px.

<div class="main__container">
    <!-- contents -->
</div>

or

<div class="main__container main__container--sm">
    <!-- contents -->
</div>

Pagination

We have overridden the default style of Bootstrap pagination to resemble more like Material Design's.


<ul class="pagination">
    <li>
        <a href="#" aria-label="Previous">
            <i class="zmdi zmdi-chevron-left"></i>
        </a>
    </li>
    <li class="active"><a href="#">1</a></li>
    <li><a href="#">2</a></li>
    <li><a href="#">3</a></li>
    <li><a href="#">4</a></li>
    <li><a href="#">5</a></li>
    <li>
        <a href="#" aria-label="Next">
            <i class="zmdi zmdi-chevron-right"></i>
        </a>
    </li>
</ul>

For lazy loading or ajax loading stuffs, you may use the alternative button-like solution.


<div class="load-more">
    <a href=""><i class="zmdi zmdi-refresh-alt"></i> Load more items</a>
</div>

Media Object

Media Object is a Bootstrap component which abstract object styles for building various types of components (like blog comments, Tweets, etc) that feature a left- or right-aligned image alongside textual content.

In Roost, default media object is customized to fit text-overflow thus media alignment feature is not functional in this template.

Basic Example

<div class="media">
   <div class="pull-left">...</div>
   <div class="pull-right">...</div>
   <div class="media-body">...</div>

Action Header

Action header is used to perform certain content level tasks such as search, sort, etc. within a page.

Front-end

<div class="action-header">
    <div class="container">

        <!-- Search -->
        <div class="action-header__item action-header__item--search">
            <form>
                <input type="text" placeholder="Search..."><!-- For desktop -->
            </form>
        </div>

        <!-- Sort -->
        <div class="action-header__item action-header__item--sort hidden-xs">
            <span class="action-header__small">Sort by :</span>

            <select class="select2">
                <option>Featured listings</option>
                <option>Newest to oldest</option>
                <option>Oldest to Newest</option>
            </select>
        </div>
    </div>
</div>

Dashboard

In dashboard, use class .action-header-alt

<div class="action-header-alt">
    <div class="container">

        <!-- Search -->
        <div class="action-header__item action-header__item--search">
            <form>
                <input type="text" placeholder="Search..."><!-- For desktop -->
            </form>
        </div>

        <!-- Sort -->
        <div class="action-header__item action-header__item--sort hidden-xs">
            <span class="action-header__small">Sort by :</span>

            <select class="select2">
                <option>Featured listings</option>
                <option>Newest to oldest</option>
                <option>Oldest to Newest</option>
            </select>
        </div>
    </div>
</div>

In addition to search and sort, you can also use the following action header items in both front-end and dashboard templates.

    <div class="action-header__item action-header__views">

Action Icons

Action Icons are used to perform a certain list of actions within a title, list or in any element using icons.


<div class="actions">
    <a href=""><i class="zmdi zmdi-favorite-outline"></i></a>
    <a href=""><i class="zmdi zmdi-print"></i></a>
    <a href=""><i class="zmdi zmdi-email"></i></a>
    <span class="dropdown">
        <a href="" data-toggle="dropdown"><i class="zmdi zmdi-share"></i></a>

        <ul class="dropdown-menu">
            <li></li>
        </ul>
    </span>
</div>

To override position of the action-icons, you can simply add a modifier class on .actions and customize it as per your need.

e.g HTML

<div class="actions actions--list">
   ...
</div>

e.g CSS

.actions--list {
    float: right;
    margin-top: 10px;
    ...
}

List Group modifier classes

  1. .list-group--condensed: Reduce left-right padding to 15px.

    <div class="list-group list-group--condensed">
    ...
</div>

  2. .list-group--bordered: Creates a 1px solid border around list-group.

    <div class="list-group list-group--bordered">
    ...
</div>

  3. .list-group--striped: Adds zebra-striping to all even list group items within list group.

    <div class="list-group list-group--striped">
    ...
</div>

  4. .list-group--block: Creates a block level list group with extra padding with other highlights (Dashboard only) .

    <div class="list-group list-group--block">
    ...
</div>
