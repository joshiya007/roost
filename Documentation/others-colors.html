Material Design Color Palette

Material Design Color Palette is a LESS/CSS toolkit for easily usage material colors by Google in websites.

This color palette comprises primary and accent colors that can be used for illustration or to develop your brand colors. Theyâ€™ve been designed to work harmoniously with each other.

For the official documentation and more examples, please visit http://zavoloklom.github.io/material-design-color-palette/index.html

Primary colors

Choose material color from below listing and use class .mdc-text-{color name} for colored text or .mdc-bg-{color name} for colored background.

<!-- Colored text -->
<span class="mdc-text-amber">Amber primary color</span>

<!-- Colored background -->
<div class="mdc-bg-amber"></div>

Other colors

Choose material color and value from colors page and use class .mdc-text-{color name}-{color value} for colored text or .mdc-bg-{color name}-{color value} for colored background.

<!-- Colored text -->
<span class="mdc-text-blue-800">Blue-800 color</span>

<!-- Colored background -->
<div class="mdc-bg-blue-800"></div>

Colored links

When you use MD Color Palette classes on links you get one step darker color on hover state.

Example: for class mdc-text-green-400 or mdc-bg-green-400 main color is green-400 and in hover state color is green-500

<!-- Colored link -->
<a class="mdc-text-green-400" href="">Green-400 color</a>

<!-- Link with colored background -->
<a class="mdc-bg-green-400" href=""></a>
