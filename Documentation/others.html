SweetAlert 2 - Custom Javascript Alerts

SweetAlert 2 is beautiful replacement for javascript alert. SweetAlert automatically centers itself on the page and looks great no matter if you're using a desktop computer, mobile or tablet. It's even highly customizeable, as you can see below!

More examples and detailed documentation at https://limonte.github.io/sweetalert2/

SweetAlert Examples

In order to use SweetAlert on your website, first you must include it's JavaScript and CSS asset files.

<!-- CSS -->
<link rel="stylesheet" href="vendors/bower_components/sweetalert/dist/sweetalert.css">

<!-- JS -->
<script src="vendors/bower_components/sweetalert/dist/sweetalert.min.js"></script>

A basic message:

swal('Any fool can use a computer')

A success message!:

swal(
'Good job!',
'You clicked the button!',
'success'
)

A title with a text under:

swal(
    'The Internet?',
    'That thing is still around?',
    'question'
    )

A warning message, with a function attached to the "Confirm"-button":

swal({
    title: 'Are you sure?',
    text: "You won't be able to revert this!",
    type: 'warning',
    showCancelButton: true,
    confirmButtonColor: '#3085d6',
    cancelButtonColor: '#d33',
    confirmButtonText: 'Yes, delete it!'
    }).then(function() {
    swal(
    'Deleted!',
    'Your file has been deleted.',
    'success'
    );
})

LightGallery - Lightbox

A customizable, modular, responsive, lightbox gallery plugin for jQuery.

The main features include responsive layout, touch support for mobile devices, youtube and vimeo support, slide and fade effects, CSS transitions with jQuery fallback, image captions and descriptions, font icon support and many more.

Official plugin site and detailed documentation at http://sachinchoolur.github.io/lightGallery/

In order to use LightGallery on your website, first you must include it's JavaScript and CSS asset files.

<!-- CSS -->
<link rel="stylesheet" href="vendors/bower_components/lightgallery/dist/css/lightgallery.min.css">

<!-- JS -->

RateYo! - Ratings

RateYo! is a tiny and flexible jQuery star rating plugin, it uses SVG to render rating, so no images required. just create a <div/>, throw some styles, initialize and thats it!.

Please visit http://rateyo.fundoocode.ninja/ for information and detailed documentation.



In order to use RateYo in your web page, make sure to include the following CSS/JS assets directly from vendors/bower_components.

<!-- CSS -->
<link rel="stylesheet" href="vendors/bower_components/rateYo/min/css/jquery.rateyo.min.css">

<!-- JS -->
<script src="vendors/bower_components/rateYo/min/js/jquery.rateyo.min.js"></script>

You can initiate RateYo ratings with any element with class of .rmd-rate.

e.g

<div class="rmd-rate"></div>

In Roost, RateYo is customized to fit more contents and you can use the following data-attributes to customize the output:

  1. data-rate-value (optional): Pre rated stars

    e.g: data-rate-value="3"

  2. data-rate-readonly (optional): Make RateYo read-only so users can't rate

    e.g: data-rate-readonly="true"

<div class="rmd-rate" data-rate-value="3" data-rate-readonly="true"></div>

jsSocials - Social Network Sharing

jsSocials is a simple social network sharing jQuery plugin. It's flexible and easily extensible. Configure visual appearance. Choose one of several themes provided. Add any yet unsupported social network if needed.

Roost uses it's own theming for jsSocial and highly recommended to use as it provided in the template.

In Roost jsSocial is used with Action Icons. You may consider adding extra styles if you want to use this plugin outside Action Icons.

For more information and detailed documentation, please head on to http://js-socials.com/



In Roost, jsSocial is initiated with class .rmd-share. You can use this class on any given element to include the basic set of sharing networks which are Facebook, Twitter, Google Plus and LinkedIn. Please refer /js/inc/functions.js to alter the default social networks and properties of this plugin.

In order to use jsSocial in your web page, make sure to include the following JS asset file right directly from /vendors/bower_components

Please note that, no need of any CSS file to include as Roost uses it's own theme.

<!-- JS -->
<script src="vendors/bower_components/jssocials/dist/js/jssocials.min.js"></script>

Basic Example

This example also uses Action Icons to show how exactly the jsSocial used in the template.

  • HTML
  • Javascript
<div class="actions">
    <div class="dropdown">
        <a href="" data-toggle="dropdown"><i class="zmdi zmdi-share"></i></a>

        <div class="dropdown-menu pull-right rmd-share">
            <div></div>
        </div>
    </div>
</div>

You can find the exact code below at /js/inc/functions.js

if($('.rmd-share')[0]) {
    $('.rmd-share > div').jsSocials({
        shares: [
            {
                share: "facebook",
                label: "",
                logo: "zmdi zmdi-facebook",
                shareIn: "blank",
                css: "rmds-item mdc-bg-indigo-400 animated bounceIn"
            },
            {
                share: "twitter",
                label: "",
                logo: "zmdi zmdi-twitter",
                shareIn: "blank",
                css: "rmds-item mdc-bg-cyan-500 animated bounceIn"
            },
            {
                share: "googleplus",
                label: "",
                logo: "zmdi zmdi-google",
                shareIn: "blank",
                css: "rmds-item mdc-bg-red-400 animated bounceIn"
            },
            {
                share: "linkedin",
                label: "",
                logo: "zmdi zmdi-linkedin",
                shareIn: "blank",
                css: "rmds-item mdc-bg-blue-600 animated bounceIn"
            }
        ]
    });
}

As you can see, this plugin is highly customizable and you can chose your icons, labels, CSS classes and pretty everything you need.

Clamp.js - Text Clamping

Clamps an HTML element by adding ellipsis to it if the content inside is too long.

In Roost Clamp.js is used to clamp overflown lines in note listing at /dashboard/notes.html

In order to use Clamp.js, you must include the following JS asset file from /vendors/bower_cpmponents

For more information and detailed documentation, head on to it's official Github repository https://github.com/josephschmitt/Clamp.js

<!-- JS -->
<script src="vendors/bower_components/Clamp.js/clamp.min.js"></script>

Sample Usage

//Single line
$clamp(myHeader, {clamp: 1});

//Multi-line
$clamp(myHeader, {clamp: 3});

//Auto-clamp based on available height
$clamp(myParagraph, {clamp: 'auto'});

//Auto-clamp based on a fixed element height
$clamp(myParagraph, {clamp: '35px'});

Usage in template:

  1. /dashboard/notes.html
