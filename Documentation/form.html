Form Groups

Form groups is used to wrap elements which are required for a single form field.

<div class="form-group">
    <input type="text" class="form-control" placeholder="Name">

    <i class="form-group__bar"></i>
</div>

Note that, <i class="form-group__bar"></i> is required inside .form-group to animate blue bar right beneath the text field.

Form group modifier classes

  1. .form-group--float: Floating label on focus.

    <div class="form-group form-group--float">
    <input type="text" class="form-control" placeholder="Name">

    <label>Name</label>

    <i class="form-group__bar"></i>
</div>

  2. .form-group--centered: Center aligned texts.

    <div class="form-group form-group--centered">
    ...
</div>

  3. .form-group--active: Shows blue bar beneath the text field.

    <div class="form-group form-group--active">
    ...
</div>

  4. .form-group--light: Used for darker backgrounds.

    <div class="form-group form-group--light">
    ...
</div>

    Usage in template:

    1. /mortgage.html
    2. /listings-list.html
    3. /blog-list.html
    4. /roost/contact.html

  5. .form-group--range: This adds extra padding on top and used to wrap input slider fields.

    <div class="form-group form-group--range">
    ...
</div>

Select 2 - Custom Selects

Select2 gives you a customizable select box with support for searching, tagging, remote data sets, infinite scrolling, and many other highly used options. Official website and detailed documentation at https://select2.github.io/


In order to use Select2, you must include JavaScript and CSS asset files on your website.

<!-- CSS -->
<link rel="stylesheet" href="vendors/bower_components/select2/dist/css/select2.min.css">

<!-- JS -->
<script src="vendors/bower_components/select2/dist/js/select2.full.min.js"></script>

Single Select

<select class="select2">
    <option value="">Single Family Home</option>
    <option value="">Condo</option>
    <option value="">Townhome</option>
    <option value="">Room</option>
</select>

You can transform any select element to a stylish select2 by simply adding the class .select2

Multiple select boxes

Select2 also supports multi-value select boxes. The select below is declared with the multiple attribute.

<select class="select2" multiple>
    <option value="" selected="">Sollicitudin</option>
    <option value="" selected="">Pellentesque</option>
    <option value="" selected="">Inceptos</option>
    <option value="" selected="">Dapibus</option>
</select>

noUiSlider - Input Slider

noUiSlider is a range slider without bloat. It offers a ton off features, and it is as small, lightweight and minimal as possible, which is great for mobile use on the many supported devices, including iPhone, iPad, Android devices & Windows (Phone) 8 desktops, tablets and all-in-ones. It works on desktops too, of course!

To read more noUiSlider and official documentation, head on to http://refreshless.com/nouislider/.


In order to use noUiSlider, you must include JavaScript and CSS asset files on your website.

<!-- CSS -->
<link rel="stylesheet" href="vendors/bower_components/nouislider/distribute/nouislider.min.css">

<!-- JS -->
<script src="vendors/bower_components/nouislider/distribute/nouislider.min.js"></script>

Basic Example

  • HTML
  • Javascript
<div id="slider-basic"></div>
var sliderBasic = document.getElementById('slider-basic');

noUiSlider.create(sliderBasic, {
	start: 100,
    range: {
        'min': 0,
        'max': 1000
    }
});

Range Slider

  • HTML
  • Javascript
<div id="slider-range"></div>
var sliderRange = document.getElementById('slider-range');

noUiSlider.create(sliderRange, {
	start: [20, 80],
	connect: true,
	range: {
    		'min': 0,
    		'max': 100
	}
});

Complete Slider with values and labels

  • HTML
  • Javascript
<div class="form-group form-group--range">
    <label>Price Range</label>
    <div class="input-slider-values clearfix">
        <div class="pull-left"><span>$</span><span id="slider-complete-upper"></span></div>
        <div class="pull-right"><span>$</span><span id="slider-complete-lower"></span></div>
    </div>
    <div id="slider-complete"></div>
</div>
var sliderComplete = document.getElementById('slider-complete');
var sliderCompleteValues = [
    document.getElementById('slider-complete-upper'),
    document.getElementById('slider-complete-lower')
]

noUiSlider.create (sliderComplete, {
    start: [12000, 70000],
    connect: true,
    range: {
        'min': 12000,
        'max': 70000
    }
});

sliderComplete.noUiSlider.on('update', function( values, handle ) {
    sliderCompleteValues[handle].innerHTML = values[handle];
});

Checkboxes and Radios

Custom checkboxes and radios are available in Roost with multiple UI variants.

Default (stacked)

<!-- Checkbox -->
<div class="checkbox">
    <label>
        <input type="checkbox">
        <i class="input-helper"></i>
        Hide Street number on listing
    </label>
</div>

<!-- Radio -->
<div class="radio">
    <label>
        <input type="radio">
        <i class="input-helper"></i>
        Hide U/Floor/Block on listing
    </label>
</div>

Basic Inline Example

Use the .checkbox-inline or .radio-inline classes on a series of checkboxes or radios for controls that appear on the same line.

<!-- Checkbox -->
<div class="checkbox checkbox-inline">
    <label>
        <input type="checkbox">
        <i class="input-helper"></i>
        1
    </label>
</div>

<!-- Radio -->
<div class="radio radio-inline">
    <label>
        <input type="radio">
        <i class="input-helper"></i>
        3
    </label>
</div>

Toggle Buttons

Add data-toggle="buttons" to a .btn-group containing checkbox or radio inputs to enable toggling in their respective styles.

<!-- Checkbox -->
<div class="btn-group" data-toggle="buttons">
    <label class="btn waves-effect">
        <input type="checkbox">1
    </label>
    <label class="btn waves-effect">
        <input type="checkbox">2
    </label>
    <label class="btn waves-effect active">
        <input type="checkbox">3
    </label>
</div>

<!-- Radio -->
<div class="btn-group" data-toggle="buttons">
    <label class="btn waves-effect">
        <input type="radio" name="options" id="option1">1
    </label>
    <label class="btn waves-effect">
        <input type="radio" name="options" id="option2">2
    </label>
    <label class="btn waves-effect active">
        <input type="radio" name="options" id="option3">3
    </label>
</div>

Character toggles (checkbox only)

Character toggles displays a character with a nice UI and on mouse hover, it's transform into a normal checkbox.

<div class="checkbox checkbox--char">
    <label>
        <input type="checkbox">
        <span class="checkbox__helper">
            <i class="mdc-bg-amber-400">A</i> <!-- You can use any color classes here -->
        </span>
    </label>
</div>

Usage in template:

  1. /submit-property.html
  2. /dashboard/index.html
  3. /dashboard/tasks-lists.html

Trumbowyg - WYSIWYG Editor

Trumbowyg is a Light, translatable and customisable jQuery plugin. Beautiful design, generates semantic code, comes with a powerful API.

In Roost, we use Trumbowyg in /dashboard/notes.html to write notes.

For more information and detailed documentation, please visit https://alex-d.github.io/Trumbowyg/

In order to use Trumbowyg, first you must include JavaScript and CSS asset files on your website.

<!-- CSS -->
<link rel="stylesheet" href="../vendors/bower_components/trumbowyg/dist/ui/trumbowyg.min.css">

<!-- JS -->
<script src="../vendors/bower_components/trumbowyg/dist/trumbowyg.min.js"></script>


  • HTML
  • Javascript
<div class="editor"></div>
$('.editor').trumbowyg();

Autosize Textarea

Autosize is a small, stand-alone script to automatically adjust textarea height.

For more information and detailed documentation, please visit http://www.jacklmoore.com/autosize/


autosize($('textarea'));
