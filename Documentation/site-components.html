Site Components

Toggleable Blocks

Toggleable Blocks are used to show/hide any elements using data-attributes set on any other elements.


In order to use this feature, you must set 2 data-attribute values to identify the click as well as the target element.

<a href="" data-rmd-actions="block-open" data-rmd-target="#sidebar">Open Sidebar</a>

When you click any element holding data-rmd-actions="block-open", it'll add class .toggled to the target element set using data-rmd-target.

Assuming #sidebar is hidden by default and in order to make it visible, we can use the following CSS

#sidebar.toggled {
   display: block
}

To reset, use data-attribute data-rmd-actions="block-close" on any clickable element.

e.g

<a href="" data-rmd-action="block-close" data-rmd-target="#sidebar">Close Sidebar</a>

This will remove .toggled class from the target element(#sidebar) set on data-rmd-target.

Source JS file: /js/inc/actions.js

Usage in template:

  1. /index.html
  2. /listings-grid.html
  3. /listings-list.html
  4. /dashboard/index.html
  5. ...

Dynamic Sidebars

This feature is used to display sidebars with different views based on the layout design.

There are 2 views available for this feature.

  1. Basic: Hidden by default, can be toggled using Toggleable Blocks
  2. Mobile: A card transforms in to a sidebar in certain screen size (992px and below).

Basic Example

<!-- Sidebar -->
<aside class="rmd-sidebar" id="sidebar">
    <!-- sidebar content -->
</aside>

<!-- Sidebar toggle button using 'Toggleable Blocks'-->
<button class="btn btn--action btn--circle" data-rmd-action="block-open" data-rmd-target="#sidebar">
    <i class="zmdi zmdi-search-for"></i>
</button>

Usage in template:

  1. /listings-list.html
  2. /listings-grid.html
  3. /agents.html

Mobile Example

<!-- Sidebar -->
<div class="rmd-sidebar-mobile" id="sidebar-mobile">
   <!-- sidebar content -->
</div>

<!-- Sidebar toggle button using 'Toggleable Blocks'-->
<!-- Make sure to display this button in 992px and below screen sizes. Note the the added responsive utility classes -->
<button class="btn btn--action btn--circle hidden-lg hidden-md" data-rmd-action="block-open" data-rmd-target="#sidebar-mobile">
    <i class="zmdi zmdi-money"></i>
</button>

Usage in template: (Resize your window to 992px or below to see the transformation)

  1. /mortgage.html
  2. /mortgage-detail-reviews.html
  3. /mortgage-detail-disclaimer.html
  4. /agent-detail.html
  5. /agent-detail-properties.html
  6. /agent-detail-reviews.html
  7. /profile-public.html
  8. /faq.html

Responsive Tabs

Transforms long list tabs in to a dropdown menu in smaller screen sizes. Responsive Tabs uses basic Bootstrap tab markup with some added wrappers. This can be also used for menu links.



In order to use responsive tabs, you must set a screen size breakpoint using data-attribute data-rmd-breakpoint.

<div class="tab-nav tab-nav--justified" data-rmd-breakpoint="650">
    <div class="tab-nav__inner">
        <ul>
            <li class="active"><a data-toggle="tab" href="#overview">Overview</a></li>
            <li><a data-toggle="tab" href="#my-searches">My Searches</a></li>
            <li><a data-toggle="tab" href="#saved-listings">Saved Listings</a></li>
            <li><a data-toggle="tab" href="#saved-agents">Saved Agents</a></li>
        </ul>
    </div>
</div>

Source JS file: /js/inc/functions.js

Usage in template:

  1. /profile.html
  2. /agent-detail.html
  3. /agent-detail-properties.html
  4. /agent-detail-reviews.html
  5. /mortgage-detail.html
  6. /mortgage-detail-reviews.html
  7. /mortgage-detail-disclaimer.html

Scroll-To

Scroll-To is a great way to smooth scroll to any DOM element.

<a href="" data-rmd-action="scroll-to" data-rmd-target="#comments" data-rmd-offset="85">
    Scroll
</a>

In order to use Scroll-To feature, you must set the following data-attributes:

  1. data-rmd-action="scroll-to"(required): Initiate scroll-to function

  2. data-rmd-target=[target class/id](required): Target element to smooth scroll

    e.g: data-rmd-target=".main-menu"

  3. data-rmd-offset=[value](optional): Adds extra offset on top to deal with fixed elements

    e.g: data-rmd-offset="100"

Usage in template:

  1. In all footers in front-end pages
  2. /blog-details.html

Icon Toggles

Icon toggles are used along with Action Icons to show active/inactive state of a certain action.

<div class="actions">
    <div class="actions__toggle">
        <input type="checkbox">
        <i class="zmdi zmdi-favorite-outline"></i>
        <i class="zmdi zmdi-favorite"></i>
    </div>
</div>

You can use any switchable icons from Material Design Icons page.
