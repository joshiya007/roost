CSS Animations

animate.css is a bunch of cool, fun, and cross-browser animations for you to use in your projects. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness.

For more information and detailed documentation, please visit https://daneden.github.io/animate.css/

Usage examples

  • CSS
  • LESS
<div class="animated [animation-name]">
    ...
</div>

e.g

<div class="animated fadeIn">
    ...
</div>

Roost also contains a custom LESS mixin which can be used in any class/ID as a property.

.animated([animation-name], [animation-duration]);

e.g

.animate {
    width: 100px;
    height: 100px;
    .animated(zoomIn, 300ms);
}
